# AlleyPin 簡報設計 skill — 安裝教學（新手友善）

這份教學把你當「完全沒用過」的新手，照著做就好。

**第一步（所有人都先做）：解壓縮**
在 Finder 找到 `alleypin-slide-design.zip`，**雙擊**它，旁邊會出現一個 `alleypin-slide-design` 資料夾。這個資料夾就是 skill 本體。

接著找你的情況：
- 用 **Claude Code**（桌面 App 或終端機）→ 方法 A（不碰指令）或方法 B（用指令）
- 用 **Chat / Cowork App** → 方法 C
- 想貼給 **ChatGPT / Magnus AI 等其他 AI** 參考 → 方法 D

---

## 方法 A — 用 Finder 裝到 Claude Code（最簡單，不碰指令）

1. 打開 **Finder**（螢幕最下面 Dock 那個藍白笑臉圖示）。
2. 按鍵盤 **⌘ Command + Shift + G**，畫面上方會跳出一個小輸入框。
3. 在框裡貼上這行字，按 **Enter**：
   ```
   ~/.claude/skills
   ```
   - 若出現「找不到資料夾」：先改貼 `~/.claude` 按 Enter，進去後在空白處按右鍵 →「新增檔案夾」→ 命名為 `skills`，再進入它。
4. 現在你人在 `skills` 資料夾裡。把第一步解壓出來的 **整個 `alleypin-slide-design` 資料夾**用滑鼠**拖進這裡**。
5. 完成！把 Claude Code 關掉重開。

## 方法 B — 用終端機裝（想用指令的人）

1. 打開「終端機」：按 **⌘ Command + 空白鍵** 開啟搜尋，輸入 `終端機`（或 `Terminal`），按 Enter。
2. 一行一行**複製貼上、每行按 Enter**：

   建立資料夾（已存在也沒關係）：
   ```
   mkdir -p ~/.claude/skills
   ```
   把資料夾搬進去（假設解壓在「下載 Downloads」）：
   ```
   mv ~/Downloads/alleypin-slide-design ~/.claude/skills/
   ```
   （若解壓在桌面，把上面的 `Downloads` 改成 `Desktop`）
3. 確認裝好：
   ```
   ls ~/.claude/skills/
   ```
   看到 `alleypin-slide-design` 就成功了。
4. 把 Claude Code 關掉重開。

## 方法 C — Chat / Cowork App

用你 App 的「skill / plugin 匯入」功能，把整個 `alleypin-slide-design` 資料夾匯入（保留裡面 `SKILL.md`、`templates/`、`assets/` 的結構不要拆散）。匯入後即可使用。

---

## 怎麼確認裝好了？

跟 Claude 說：
> 「幫我做一份關於『新功能介紹』的 AlleyPin 簡報」

它應該會**先問你「要 PPTX 離線檔還是 Google 簡報」**，然後用 AlleyPin 的藍＋橘品牌色做出來。有這個反應，就是裝好了。

---

## 方法 D — 貼給 ChatGPT / Magnus AI 等其他 AI 參考

這些工具沒辦法「安裝」skill，但可以「參考」它來做：

1. 用「文字編輯 / 記事本」打開 **`SKILL.md`**，**全選（⌘A）→ 複製（⌘C）**。
2. 到 ChatGPT / Magnus 的對話框，**在最前面貼上**，並加一句話：
   > 「以下是 AlleyPin 簡報設計規範。請嚴格依照它製作我接下來要的簡報：用 HTML、1280×720、遵守書寫規範，並先問我要 PPTX 還是 Google 簡報。」
3. 需要版型時，把 `templates/` 裡對應的 `.html`（例如 `cover-a.html`、`content-rows.html`）內容也一起貼進去，當作起點。
4. 若該工具能上傳圖片，把 `assets/` 裡的 logo 一起上傳。

💡 也可以把 `SKILL.md` 全文放進 ChatGPT 的「自訂指令（Custom Instructions）」，這樣每次對話都會自動套用。

## 字體說明

- **`Noto Sans TC`**：一般電腦、Google 簡報都內建，不用另外裝。
- **`LINE Seed TW`**：只有要做「PPTX 離線檔、且想用品牌標題字」時才需要。到 **seed.line.me** 免費下載安裝即可（開源、可商用）。若沒裝，標題就用 Noto Sans TC，不影響使用。
